# BOM
1× NE555P (DIP-8), 1× CD4017BE (DIP-16), 10× 5 mm LEDs (D1–D10), 1× 1 kΩ resistor (R1), 1× 470 Ω resistor (R2), 1× 50 kΩ potentiometer (RV1), 1× 0.01 µF ceramic capacitor (C1), 1× 1 µF electrolytic capacitor (C2), 2× M3 mounting holes (H1, H2).
